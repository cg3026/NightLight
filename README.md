# NightLight : 夜间灯光遥感数据处理

### 该程序的目的在于自动化实现：
 >+ 日夜间灯光数据的自动获取。 
 >+ 灯光数据的自动校正。 
 >+ 基于灯光数据的服务发布与提供。
 
 在全部自动化流程中存在四个部分内容，根据执行时间顺序依次为，`DMSP年度灯光数据校正`，`NPP年度灯光数据校正`，`NPP月度灯光数据校正`，`NPP每日灯光数据爬取`，
`NPP每日灯光数据校正`，`灯光数据服务发布`

### 文件结构
> 文件下属五个文件夹： \
> `DailyLight`: 实现日度数据的自动化校正。 \
> `DMSP`:实现1997-2013年度DMSP年度数据的全流程校正，只执行一次，1997-2013年度灯光数据作为成熟产品提供。 \
> `NPP`:实现2013年及以后NPP年度数据的全流程校正，需要跟随年份进行更新。 \
> `Spyder`:实现对日数据的定时爬取，内部包含定时任务模块，设定为每晚进行爬取，间隔为24h。\
> `Publish`:将处理完的数据进行商品加工，由于过于遥远，暂时并未建立
